# Obhodiq

[English](README.md) | [Русский](README.ru.md)

Obhodiq — это beta-дополнение для [Podkop](https://github.com/itdoginfo/podkop) на OpenWrt.

Сейчас большинство VPN-продавцов выдают не готовый конфиг, а ссылку-подписку. Obhodiq сделан именно под такой сценарий: он берёт подписку, разбирает её, собирает список серверов и передаёт результат в Podkop, чтобы дальше можно было оставить `URLTest` или выбрать сервер вручную.

> [!IMPORTANT]
> Obhodiq **не заменяет** Podkop. Он работает **вместе** с оригинальным Podkop и требует, чтобы Podkop был установлен заранее.

> [!WARNING]
> Сейчас Obhodiq находится в **beta-статусе**. Разные провайдеры и разные типы подписок могут вести себя по-разному. Пользователи этой версии по сути тоже выступают бета-тестерами.

## Что делает Obhodiq

- принимает VPN-подписки по ссылке
- загружает и разбирает поддерживаемые форматы подписок
- собирает список серверов для Podkop
- оставляет автоматический режим `URLTest`
- даёт ручной выбор сервера
- позволяет включать и выключать отдельные серверы
- показывает информацию о подписке, текущем сервере и пинге от Podkop
- умеет обновлять подписку по расписанию

## Как это работает

1. Добавляешь ссылку подписки.
2. Obhodiq скачивает подписку и разбирает поддерживаемые ссылки внутри неё.
3. Поддерживаемые серверы подготавливаются и передаются в Podkop.
4. Дальше Podkop уже занимается маршрутизацией, `URLTest` и проверкой задержки.

## Форматы подписок

Obhodiq рассчитан на те подписки, которые сейчас чаще всего выдают VPN-провайдеры: в первую очередь это V2Ray / Xray / sing-box подписки и провайдерские обёртки вокруг них.

Сейчас парсер умеет работать с:

- обычными списками ссылок
- base64-подписками
- многими JSON-подписками
- HAPP-обёртками вида `happ://add/https://...`

Семейства ссылок, которые сейчас умеет обрабатывать парсер:

- `vless://`
- `vmess://`
- `trojan://`
- `ss://`
- `socks4://`
- `socks5://`
- `hy2://`
- `hysteria://`
- `hysteria2://`

## Совместимость

Obhodiq разбирает подписку и подготавливает список серверов, но уже сам Podkop решает, что он реально может запустить, пропинговать и использовать.

Что сейчас жёстко отсекается до экспорта:

- `XHTTP`

Дополнительно:

- `happ://add/https://...` у нас считается форматом-обёрткой, а не отдельным типом прокси
- шифрованные `happ://crypt4/...` — это отдельный случай, и здесь они не заявляются как полностью поддержанные
- `WS`, `GRPC`, `Hysteria` и похожие форматы могут нормально распарситься, но потом не заработать уже на стороне Podkop или у конкретного провайдера
- большая часть того, что Podkop нормально переваривает, должна работать и здесь, но не каждый импортированный сервер гарантированно заработает

## Требования

- OpenWrt
- уже установленный оригинальный [Podkop](https://github.com/itdoginfo/podkop)
- рекомендуемые версии Podkop: `0.7.19`, `0.7.20`
- рекомендуемые версии OpenWrt: `24.10.6`, `25.12.5`

## Установка

Сначала установи оригинальный Podkop:

```sh
sh <(wget -O - https://raw.githubusercontent.com/itdoginfo/podkop/refs/heads/main/install.sh)
```

Потом установи Obhodiq:

```sh
sh <(wget -O - https://raw.githubusercontent.com/renqismike/obhodiq/main/install.sh)
```

## Ручная установка

Если нужна ручная установка, используй файлы пакетов из release assets или из папки `dist/` в репозитории.

Для OpenWrt с `opkg`:

```sh
opkg install obhodiq_0.1.0-r2_all.ipk luci-app-obhodiq_0.1.0-r2_all.ipk
```

Для OpenWrt с `apk`:

```sh
apk add --allow-untrusted obhodiq-0.1.0-r2.apk luci-app-obhodiq-0.1.0-r2.apk
```

## Удаление

```sh
sh uninstall.sh
```

Или вручную через `opkg`:

```sh
opkg remove luci-app-obhodiq
opkg remove obhodiq
```

Или вручную через `apk`:

```sh
apk del luci-app-obhodiq
apk del obhodiq
```

Если нужно гарантированно убрать и сохраненную ссылку подписки, используйте `uninstall.sh` из релиза.

Obhodiq должен удаляться без удаления самого Podkop.

## Интерфейс

Основные действия:

- `Save URL / Сохранить ссылку` — сохраняет ссылку подписки
- `Refresh subscription / Обновить подписку` — заново скачивает и пересобирает текущую подписку
- `Auto update / Автообновление` — задаёт расписание автообновления
- кнопка питания — включает или отключает интеграцию Obhodiq

Список серверов:

- `Auto / Авто` оставляет Podkop в режиме `URLTest`
- радиокнопка переключает между авто и ручным сервером
- галочка включает или исключает сервер из экспорта
- `Ping / Пинг` показывает значение, которое вернул Podkop

## Статус проверки

Это всё ещё beta-версия, и она проверялась только на ограниченном количестве подписок от разных VPN-провайдеров, а не на всех возможных вариантах подписок на рынке.

Во время разработки приложение проверялось на:

- OpenWrt `24.10.6` с `opkg`
- OpenWrt `25.12.5` с `apk`
- Podkop `0.7.19-r1`
- Podkop `0.7.20-r1`
